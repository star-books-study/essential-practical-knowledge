# 8장. 실무에서 꼭 필요한 보안 지식
## HMAC 을 이용한 데이터 검증
- 메시지가 위변조되지 않았음을 확인할 수단으로 HMAC (Hash-based Message Authentication Code) 을 주로 사용
  - HMAC : 메시지 무결성 (위변조되지 않은 메시지) + 인증 (메시지 발신자 인증) 보장하기 위해 사용하는 암호화 기술 
  - 해시 함수와 비밀키 이용 (발신자, 수신자만 알고 있어야 함)
- 메시지를 비밀키로 해싱하여 MAC (메시지 인증 코드) 생성 -> 원본 메시지와 함께 수신자에게 전송 -> 수신자는 수신한 메시지에 대해 비밀키로 MAC 을 생성하고 발신자가 보낸 MAC 과 비교
